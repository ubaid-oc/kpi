Key UI Elements

NL Input Bar
Top of panel. Type a description, click Generate. Replaces current expression with AI-generated AST.
Tab Bar
Switch between Calculation, Validation, and Skip Logic for the selected field. Each tab has its own expression.
Structured Rows
Each row shows a sub-expression with colored tokens. Click edit to expand inline dropdowns for field/operator/value.
Raw Expression
Collapsible section showing XPath. Toggle enables direct text editing with bidirectional sync to visual builder.
